
# FullCalendar Premium TypeScript Example

This is a fully-buildable example project for [FullCalendar Premium](https://fullcalendar.io/pricing) and [TypeScript](https://www.typescriptlang.org/), leveraging [Vite](https://vitejs.dev/).


## Installation

```bash
git clone https://github.com/fullcalendar/fullcalendar-examples.git
cd fullcalendar-examples/typescript-scheduler
npm install
```

## Build Commands

```bash
npm run dev   # builds and opens a web browser
npm run build # builds files into dist/ directory
npm run clean # start fresh
```
